# Climb_and_rest
```
【製作期間】	3ヶ月(実作業時間50h程)
【使用言語】	C言語
【ジャンル】	テキストオンリーな登山ゲーム
【概要】	山を登ります。HPが無くならないように休憩しながら
		上を目指していきます。
【操作方法】	wキーで登り、sキーで下ります。
【ゲームクリア】頂上（５００地点）にたどり着いたらクリアーです。
【書いた範囲】	Ver.2013.cの全て
【制作動機】	とても楽しかったゲームをまたやろうと思い探したところ、
		見つからなかったので似たようなものを自分で作りました。
【頑張った所】	成長システムと１画面に納めたところです。
【面白いところ】ダメージが計算できるのでギリギリのところを行くのも良し、
		余裕をもって安全マージンを取るも良し、といった
		自分なりの戦略が考えられるところです。
```
